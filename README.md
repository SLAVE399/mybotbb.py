📋 Manual Setup
1. Get API Keys
API_ID & API_HASH → my.telegram.org
BOT_TOKEN → @BotFather
2. Fill in bot.py
Open bot.py and replace:
API_ID = 12345678
API_HASH = "your_api_hash_here"
BOT_TOKEN = "your_bot_token_here"
3. Install & Run
pip install -r requirements.txt
python bot.py
